
# My Personal Expense Tracker

A simple SQL database I made to track my daily expenses.

## What it does
- Stores my expenses with categories
- Shows total spending
- Tells me which category I spent the most on

## How to run
1. Open MySQL Workbench
2. Copy expense_tracker.sql
3. Run it

## Files
- expense_tracker.sql - main database file

## What I learned
- Basic SQL commands
- How to use foreign keys
- Grouping data with GROUP BY
